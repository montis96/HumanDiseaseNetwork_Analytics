Struttura del progetto:

- Cytoscape: contiene le reti costruite con cytoscape
	- HDN-full: rete completa di tutte le analisi
	- HDN-toWeb: rete utilizzata per la creazione della demo

- Dataset: dataset utilizzati per le analisi post preprocessing
	- Original: cartella contenente i dataset original prima del preprocessing

- Demo: demo interattiva della rete (html, javascript)

- Documents: documenti letti per lo svolgimento del progetto

- Notebook: script in python utilizzati per il preprocessing

- PaperHDN: Paper sulle analisi svolte sulla rete
